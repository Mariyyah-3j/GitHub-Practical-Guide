git

# 🟣 المرحلة الثالثة: أول مشروع باستخدام Git
 

🎯 الهدف:
بناء “آلة زمن” لمشروعك.
كل commit = لقطة محفوظة تقدر ترجع  لها وتتابع  التغييرات.

1) تفعيل Git داخل المشروع:
git init
git status

2) إضافة الملفات:
ملف محدد:
git add index.html

كل الملفات:
git add .

3) حفظ التعديلات:
git commit -m "Add login page"

قاعدة رسالة الـ commit:
فعل + شيء محدد (Add / Fix / Update)

4) مراجعة السجل:
git log --oneline

🔁 الروتين الذهبي عند أي تعديل:
git status
git diff
git add .
git commit -m "Describe change"

⚠️ تنبيه:
قبل git add دائمًا تأكد  من التغييرات بـ:
git diff
